## how this project was built

```sh
mkdir netcore-psql-util
cd netcore-psql-util

dotnet new sln
dotnet new classlib -n netcore-psql-util

dotnet sln netcore-psql-util.sln add netcore-psql-util/netcore-psql-util.csproj
dotnet restore
dotnet build
```
